# LinkVault - Your Personal Link Organizer
**Live Site (https://linkvault-organizer.netlify.app/)

**LinkVault** is a modern, feature-rich web application designed to help you organize, manage, and access your favorite websites and bookmarks efficiently. Built with pure HTML, CSS, and JavaScript, this application provides a sleek, intuitive interface for managing your digital resources with local storage persistence.

## üåü Features

### Core Functionality
- **Add Links**: Easily add new links with title, URL, category, and description
- **Edit Links**: Modify existing links with an intuitive editing interface
- **Delete Links**: Remove unwanted links with confirmation dialog
- **Categorization**: Organize links into categories (Learning, Work, Personal)
- **Favorites System**: Star your most important links for quick access
- **Search & Filter**: Find links quickly with real-time search and category filtering

### Import & Export Capabilities
- **Multiple Import Options**:
  - Browser bookmarks (HTML files from Chrome, Firefox, Edge, Safari)
  - JSON files with structured link data
  - CSV files (spreadsheet format)
  - Plain text files with URLs
  - Direct URL pasting
- **Multiple Export Options**:
  - JSON format (perfect for backup and re-importing)
  - Plain text format
  - CSV format (for spreadsheets)
  - HTML bookmarks (browser-compatible format)

### User Experience
- **Responsive Design**: Works seamlessly on desktops, tablets, and mobile devices
- **Dark Theme**: Sleek black and white color scheme with subtle animations
- **Real-time Statistics**: Visual counters for total links, favorites, and categories
- **Toast Notifications**: User-friendly feedback system
- **Keyboard Shortcuts**: Ctrl+K (search), Ctrl+N (add link), Ctrl+I (import), Ctrl+E (export)
- **Loading Indicators**: Visual feedback during import/export operations

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser

### Installation
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start organizing your links immediately!

### Usage
1. **Add a Link**: Click "Add Link" and fill in the details
2. **Browse Categories**: Use the category tabs to filter your links
3. **Search**: Use the search bar to find specific links
4. **Manage**: Use the action buttons to edit, delete, or favorite links
5. **Import/Export**: Use the Import and Export buttons to manage your data
